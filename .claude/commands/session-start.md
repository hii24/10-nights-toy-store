Выполни ритуал начала сессии (методология §4.2). Ничего не реализуй на этом шаге, только подготовься:

1. `pwd` — подтверди рабочую директорию.
2. Прочитай `harness/claude-progress.md` и выполни `git log --oneline -20`.
3. Прочитай `harness/feature_list.json`. Выбери ОДНУ фичу с `"passes": false` наивысшего приоритета для текущей фазы.
4. Запусти `harness/init.sh` и сделай базовый смоук-тест (игра не сломана?). Если что-то сломано — сначала почини это, прежде чем брать новую фичу.
5. Выведи: текущее состояние проекта, выбранную фичу, и предложи план (в plan mode, Opus) её реализации. Дождись моего «ок» перед кодом.

Помни скоуп-гардрейлы из `.claude/CLAUDE.md`: только текущая фаза.
