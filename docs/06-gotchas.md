# 06 — Грабли и затыки (читать внимательно)

## Данные / ProfileStore
- DataStore НЕ работает в локальном тесте Studio по умолчанию — включить API access или mock.
- При быстром реджоине профиль грузится до ~7с (сессионный лок) — обрабатывать.
- НЕ входить с дефолтными данными при сбое загрузки (перезапишет сейв).

## Аудио (Suno + Roblox)
- Suno = музыка, не SFX. SFX — из Roblox audio library / ElevenLabs.
- Коммерческие права Suno — только платный тариф; статус AI-музыки не устаканен; модели deprecated в 2026 -> бэкапить.
- Roblox: лимит 2000/30 дней только при верификации ID (иначе 100) -> верифицировать заранее. Модерация — очередь. Asset privacy — выдать разрешение опыту. Уложиться в <=7мин/<=20MB/<=48kHz.

## 3D-генерация (Cube)
- Выход требует чистки; тоглы EditableImage/EditableMesh обязательны; rate-limit ~5/мин.
- Смешение генерённого/купленного разваливает стиль -> прогон через арт-библию.

## Геймплей / движок
- StreamingEnabled + серверные враги: части могут быть не загружены на клиенте -> логику врагов держать на сервере, аккуратно стримить важные объекты.
- PathfindingService дорог и дёргается на многих агентах -> для MVP waypoint-ноды, реальный pathfinding только при необходимости.

## Claude Code / агенты («не сломаться на пол пути»)
- Без harness агент one-shot'ит и теряет контекст / объявляет победу рано -> строго feature_list.json + progress + init.sh + ритуал сессии.
- Агент может галлюцинировать API/писать по устаревшему -> Context7 + CLAUDE.md + тесты как защита.
- Лимиты Claude Code общие (Code+чат+Cowork) и урезаны в пик (5-11 утра PT) -> планировать тяжёлые многоагентные прогоны.
- MCP даёт исполнение кода в Studio -> ревью, read-only для незнакомых серверов.
- Open Cloud Luau Execution + Mantle+AWS-state — заметная разовая настройка -> вводить поэтапно.

## Продукт / лонч
- Roblox холодный старт: «тихий» запуск может не получить данных -> внешний толчок (thumbnails/видео/Discord/TikTok).
- Мобильную производительность и читаемость тестировать на реальном телефоне.
- Реальный плейтест с людьми — рано и часто; аналитика покажет, где отваливаются в первой минуте.
